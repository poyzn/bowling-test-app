### Stack

* Ruby version 2.7.1
* Ruby on Rails 6.1
* Redis 

### Run locally

1. Make sure required Ruby version is installed.
2. Make sure Redis server is installed and running.
3. Run `bundle install` to install gems.
4. Run `rspec`
